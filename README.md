# 🚀 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website crafted to showcase software engineering projects, data engineering workflows, and full-stack development capabilities. Built with performance, clean UI/UX, and seamless interactivity in mind.

---

## ✨ Features

- **📱 Fully Responsive Design:** Fluid layouts optimized for all screen sizes, from mobile devices to ultra-wide desktop displays.
- **🎡 Interactive Project Showcases:** Smooth, swipeable project carousels powered by **Swiper.js** with customized pagination and slide transitions.
- **✨ Dynamic Animations & Micro-interactions:** Engaging scroll animations and element entrances implemented with **ScrollReveal** and **Anime.js**.
- **🎯 Clean Typography & Layout:** Modern CSS architecture using CSS logical properties (`padding-block`, Flexbox/Grid) for robust structural styling.
- **⚡ Fast Performance:** Lightweight frontend structure built with vanilla technologies and modular asset loading.

---

## 🛠️ Tech Stack

### Frontend & UI
- **HTML5:** Semantic markup ensuring accessibility and clean document structure.
- **CSS3:** Custom styling, responsive media queries, and modern layout techniques.
- **JavaScript (ES6+):** Core DOM manipulation, interactive components, and state management.

### Libraries & Tools
- **[Swiper.js](https://swiperjs.com/):** Touch-enabled interactive sliders for the projects section (`.projects__swiper`).
- **[ScrollReveal](https://scrollrevealjs.org/):** Scroll-driven reveal animations for section headers and cards.
- **[Anime.js](https://animejs.com/):** Lightweight JavaScript animation engine for UI micro-interactions.
